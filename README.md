# CS202_ComOrgProject

This repository is for a Computer Organization course project. The goal is to design and implement a simple pipelined CPU in Verilog.

## Project Goals

- Implement the CPU datapath and control logic in Verilog
- Build out the pipeline stages step by step, including fetch, decode, execute, memory, and write-back
- Verify correctness through simulation and iterate on the design

## Current Status

The project is in its initial setup stage. The following items will be added over time:

- Source code structure
- Instruction set and functional notes
- Simulation test cases
- Design documentation and lab records

## Usage

When simulation or waveform viewing is needed, follow the project documentation and test scripts that will be added later.

## Planned Structure

Recommended repository layout:

- rtl/ for Verilog source code
- tb/ for testbenches and simulation code
- docs/ for design notes, lab reports, and documentation
- scripts/ for helper scripts
